## Featured Projects

### Pharmaceutical Inventory Demand Forecasting using Machine Learning · [Live App](https://pharmainsight.streamlit.app/)

A demand forecasting pipeline for pharmaceutical inventory management, built on real-world stock records across multiple healthcare facilities and states.

**What it does**

- Engineers **lag-based (1-3 month)** and **rolling-window** features from 14,700+ stock records across 49 medicines and 25 healthcare facilities in 5 states, capturing temporal consumption and depletion patterns
- Implements a **Random Forest regression model**, optimized with RFE for feature selection and GridSearchCV for hyperparameter tuning, achieving **R² = 0.95 (95.37% accuracy)** in predicting closing stock levels
- Surfaces top predictive features (consumption, lag-stock, rolling-mean) via feature importance analysis
- Deployed as an interactive **Streamlit web app**

**Tech:** Python, Pandas, Scikit-learn, Random Forest, RFE, GridSearchCV, Streamlit

Repo: [Pharmaceutical-Inventory-Demand-Forecasting-using-Machine-Learning](https://github.com/hamzaikram2026/Pharmaceutical-Inventory-Demand-Forecasting-using-Machine-Learning)

### AutoInsight - Automated EDA & Predictive Modeling Pipeline · [Live App](https://autoinsight-26.streamlit.app/)

An AI-powered data analysis platform — upload any CSV, ask a business question, and get automated profiling, cleaning, baseline ML models, and a full report.

**What it does**

- Architects an end-to-end automated pipeline — validation, cleaning, profiling, target/task inference, modeling, and reporting
- Auto-infers the prediction target and task type (classification vs. regression) directly from a natural-language business question and dataset schema
- Runs a Scikit-learn preprocessing pipeline (imputation, scaling, one-hot encoding) to train and benchmark baseline models (Logistic Regression, Random Forest) with task-appropriate metrics (F1, accuracy, R², MAE, RMSE)
- Auto-generates diagnostic visualizations (missingness, distributions, correlation heatmaps) and a downloadable executive markdown report, with an extensible layer for LLM-based narrative generation

**Tech:** Python, Streamlit, Pandas, Scikit-learn, Plotly

Repo: [AutoInsight-Automated_EDA-Predictive_Modeling_Pipeline](https://github.com/hamzaikram2026/AutoInsight-Automated_EDA-Predictive_Modeling_Pipeline)
